## 📌 Project Overview

> **The financial services industry faces a mounting challenge — traditional loan recovery methods are failing at scale.**
> Industry data confirms that conventional human-agent-driven recovery methods yield success rates **below 20%**, while the global debt collection software market — valued at **USD 4.92 billion in 2023** — grows at a CAGR of **9.6%**, signalling urgent demand for smarter alternatives.

This research proposes an **AI-Based Smart Loan Recovery Calling System** — an integrated framework leveraging:

<div align="center">

<table>
  <thead>
    <tr>
      <th>🔧 Technology</th>
      <th>💡 Purpose</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>🗣️ Natural Language Processing (NLP)</td>
      <td>Understand borrower intent across 10+ languages and 500+ dialects</td>
    </tr>
    <tr>
      <td>🎙️ ASR + TTS</td>
      <td>Real-time speech-to-text and text-to-speech conversion</td>
    </tr>
    <tr>
      <td>🧠 Sentiment Analysis (BERT / LLM)</td>
      <td>Detect distress, anger, and cooperation in real time</td>
    </tr>
    <tr>
      <td>📊 Predictive Analytics (XGBoost, LightGBM)</td>
      <td>Repayment Propensity Score per borrower</td>
    </tr>
    <tr>
      <td>⚖️ RBI Compliance Engine</td>
      <td>Contact windows, disclosure mandates, audit logging enforced by design</td>
    </tr>
    <tr>
      <td>🤖 ML Pipeline</td>
      <td>Continuous model improvement with chronological train-test splits</td>
    </tr>
  </tbody>
</table>

## 📞 AI Call Flow

```
  📲 CALL INITIATED
        │
        ▼
  ╔═══════════════════════════════════════════════════════════╗
  ║ STEP 1  │  🆔 Identification & Disclosure                 ║
  ║         │  AI identifies itself + lending institution     ║
  ║         │  (RBI mandatory requirement)                    ║
  ╚═══════════════════════════════════════════════════════════╝
        │
        ▼
  ╔═══════════════════════════════════════════════════════════╗
  ║ STEP 2  │  🔍 Intent Probing                              ║
  ║         │  Bot inquires about overdue awareness           ║
  ║         │  + current payment ability                      ║
  ╚═══════════════════════════════════════════════════════════╝
        │
        ▼
  ╔═══════════════════════════════════════════════════════════╗
  ║ STEP 3  │  💰 Personalised Offer Presentation             ║
  ║         │  Repayment options based on RPS + segment       ║
  ║         │  (lack of funds / forgetfulness / wilful)       ║
  ╚═══════════════════════════════════════════════════════════╝
        │
        ▼
  ╔═══════════════════════════════════════════════════════════╗
  ║ STEP 4  │  ✅ Payment Facilitation                        ║
  ║         │  UPI link sent via SMS / WhatsApp               ║
  ║         │  Promise to Pay (PTP) recorded                  ║
  ╚═══════════════════════════════════════════════════════════╝
        │
        ▼
  ╔═══════════════════════════════════════════════════════════╗
  ║ STEP 5  │  🚨 Escalation (if needed)                      ║
  ║         │  Disputes / distress → Human Agent              ║
  ║         │  Full conversation summary transferred          ║
  ╚═══════════════════════════════════════════════════════════╝
```
